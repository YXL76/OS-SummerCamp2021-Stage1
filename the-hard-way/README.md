# The Hard Way

基于笨办法学 Python ，重新实现 15 道编程练习题。

运行某一练习：

```shell
cargo run --bin ex4
```

测试某一练习：

```shell
cargo test --bin ex4
```

测试所有练习：

```shell
cargo test --bins
```
